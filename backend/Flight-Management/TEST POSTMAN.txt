CREAR VUELO:

curl -X POST http://localhost:5031/flights \
     -H "Content-Type: application/json" \
     -d '{
           "origin": "New York",
           "destination": "Los Angeles",
           "departure_time": "2025-02-01T12:00:00",
           "arrival_time": "2025-02-01T15:30:00",
           "price": 250.50,
           "status": "active",
           "total_seats": 200,
           "available_seats": 200
         }'


OBTENER TODOS LOS VUELOS:

curl -X GET http://localhost:5032/flights


OBTENER VUELO POR ID :
curl -X GET http://localhost:5033/flights/1


MODIFICAR VUELO:

curl -X PUT http://localhost:5034/flights/1 \
     -H "Content-Type: application/json" \
     -d '{
           "status": "delayed",
           "available_seats": 180
         }'


BORRAR VUELO:

curl -X DELETE http://localhost:5035/flights/1
