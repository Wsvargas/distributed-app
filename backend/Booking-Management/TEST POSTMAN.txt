CREAR RESERVA:
curl -X POST http://localhost:5021/booking \
     -H "Content-Type: application/json" \
     -d '{
           "user_id": 100,
           "flight_id": 100,
           "booking_date": "2025-02-01T14:30:00",
           "status": "confirmed"
         }'


OBTENER TODAS LAS  RESERVAS:

curl -X GET http://localhost:5022/booking



ACTUALIZAR RESERVA:
curl -X PUT http://localhost:5023/booking/5 \
     -H "Content-Type: application/json" \
     -d '{
           "user_id": 1,
           "flight_id": 101,
           "status": "cancelled"
         }'


ELIMINAR RESERVA:
curl -X DELETE http://localhost:5024/booking/5
